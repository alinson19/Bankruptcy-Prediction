**Bankruptcy Prediction Using Machine Learning**

This project develops a machine learning model to predict the likelihood of corporate bankruptcy based on financial indicators. The dataset contains multiple financial ratios and company attributes that help determine whether a company is at risk of bankruptcy.

The model was built using the Random Forest algorithm and trained on a dataset that includes both bankrupt and non-bankrupt companies. Since the dataset was imbalanced, SMOTE (Synthetic Minority Over-sampling Technique) was applied to balance the classes and improve the model's ability to detect bankruptcy cases. Feature importance analysis was also performed to identify the most influential financial indicators contributing to bankruptcy prediction.

The trained model was deployed as an interactive web application using Streamlit. The application allows users to input financial features and receive a prediction indicating whether a company is likely to go bankrupt, along with the probability of bankruptcy risk.

The project demonstrates the practical use of machine learning in financial risk analysis and shows how predictive models can assist investors, analysts, and financial institutions in identifying potential bankruptcy risks.
